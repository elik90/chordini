Run Chordini_NB in jupyter.  The notebook uses table of contents
The notebook will create the following files:

	X_ml.csv - used in tableau

	option1 - to convert using prog_player.py

	km_cluster_center_chords - to convert using prog_player.py

To convert files into musescore scores, run:
python prog_player <filename>

This will automatically open musescore for audible formats.  
To export as mp3 files, navigate to file -> export
